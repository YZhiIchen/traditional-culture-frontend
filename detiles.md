### 技术栈

| 层 | 技术 | 说明 |
|---|------|------|
| 框架 | Vue 3 (Composition API) | `<script setup>` 写法 |
| 语言 | TypeScript | 严格模式 |
| 构建 | Vite 6 | 开发端口 3000，代理 `/api` → `localhost:8080` |
| 状态管理 | Pinia | `app` + `user` 两个 Store |
| 路由 | Vue Router 4 | Hash 模式 + 路由守卫 |
| UI 组件 | Element Plus | 全局注册 + 图标全量引入 |
| 图表 | ECharts 5 | 识别详情页标签分布图 |
| HTTP | Axios | 自定义 `Request` 封装类 |
| 样式 | SCSS + CSS Variables | 当代山水色彩体系 |
